## Jquery 
1. Nhúng jquery vào trang html 
- Tải về từ trang chủ : jquery.com, ...
    <script src ="js/jquery.min.js"></script>
- Nhúng trực tiếp từ cloud 
    <script src = ""></script>

2. Cú pháp 
    $(selector).action({

    })
*selector: 
    - tên thẻ html: h1, p, div, ...
    - tên class: .test, .box, ...
    - tên id: #test, #zone1,...
    - giá trị thuộc tính: input[type='text];
    - cú pháp css selector: div>p, #zone1 p, ...
* action: các hoạt động trên selector
    click, hide, show, hover, mouseover, blur, scroll, resize,...
